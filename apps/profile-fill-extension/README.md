# Profile Fill Assistant

Chrome Manifest V3 extension for saving a small profile and filling common job application forms from the active tab.

## Load in Chrome

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked.
4. Select this folder:
   `apps/profile-fill-extension`

## Use

1. Open the extension popup.
2. Fill in your profile fields.
3. Click Save profile.
4. Open a job application page.
5. Click Fill current page.

Profile data stays in Chrome extension local storage.
