# StyleMirror Roadmap

## MVP

- Camera-first React Native app shell
- Mock face scan result
- Face-shape and hairline-aware recommendations
- Celebrity reference cards
- Manual hairstyle overlay placeholder

## AI Integration

1. Face landmarks from camera or uploaded image.
2. Face measurements derived from landmarks.
3. Rule-based face-shape classification.
4. Hairline stage estimation from guided front photo.
5. Hairstyle positioning using forehead, temple, ear, and chin landmarks.

## Guardrails

- Treat hair loss output as style guidance, not diagnosis.
- Use language like "appears", "possible", and "style match".
- Do not claim the user needs a transplant.
- Keep face images private and process on-device where possible.

## Later

- Beard and glasses try-on
- Women style catalog
- Clothes try-on
- Salon/barber reference mode
- Clinic education mode for transplant previews
